## 🎯 如何保存图片

**方法 1: 直接在聊天中下载**
1. 右键点击聊天中的每张图片
2. 选择「另存为」或「Save Image As」
3. 使用上面列出的对应文件名保存
4. 将文件上传到 `pilgrimage-journey/public/images/` 文件夹中

**方法 2: 使用文件管理器**
1. 在 Same 的文件管理器中打开 `pilgrimage-journey/public/images/` 文件夹
2. 点击「上传文件」按钮
3. 选择您保存的图片并上传

## 📐 图片建议

- **格式**：JPG 或 WebP
- **分辨率**：1920x1080 或更高（支持高清显示）
- **文件大小**：每张 < 800KB（优化后性能更好）
- **质量**：80-90%
- **宽高比**：16:9 最佳
